Mastermind

Instructions

The purpose of Mastermind is to figure out the secret numeric code set by the computer. The length of the code and maximum digit allowed in each slot is indicated in the settings menu below. Enter your guess and click Submit. A valid guess can be 1234, 1122, or any combination of digits that fits the game settings. Each time you submit a guess, you'll be given feedback indicating how close you are. White dots indicate how many digits in your guess appear in the secret code but in an incorrect position. Black dots indicate how many digits in your guess appear in the secret code and in the correct position.
